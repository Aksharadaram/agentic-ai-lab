# Assignment 1: Rule-Based AI Agent

## Objective

To build a simple AI agent using rule-based logic.

---

## Architecture

```
Input → Decision → Action
```

---

## Components

* `input_handler.py` → Handles user input
* `decision_engine.py` → Determines intent using keywords
* `actions.py` → Executes actions
* `agent.py` → Orchestrates the workflow

---

## Features

* Greeting response
* Date retrieval
* Basic arithmetic calculation

---

## Limitations

* Fragile keyword matching
* No understanding of natural language
* Not scalable for complex tasks

---

## Learning Outcome

* Understanding of basic agent pipeline
* Importance of modular design
